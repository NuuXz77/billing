### Query untuk melihat transaksi per member:

```sql
-- Lihat semua transaksi Member 3
SELECT id, user_id, user_transaction_number, transaction_code
FROM transactions
WHERE user_id = 3
ORDER BY user_transaction_number ASC;

-- Lihat nomor transaksi terakhir Member 3
SELECT MAX(user_transaction_number) as last_number
FROM transactions
WHERE user_id = 3;

-- Lihat semua member dan jumlah transaksinya
SELECT 
    users.id,
    users.full_name,
    COUNT(transactions.id) as total_transactions,
    MAX(transactions.user_transaction_number) as last_transaction_number
FROM users
LEFT JOIN transactions ON users.id = transactions.user_id
WHERE users.role = 'member'
GROUP BY users.id, users.full_name
ORDER BY users.id;
```
